Kokborok Numerals

Lekhamung — the Kokborok counting system. This page presents the numerals using the Hachukma script. Numbers are shown with Arabic numerals, Hachukma digit representations, the Kokborok numeral, its representation in Hachukma PUA characters, and Roman numerals (where applicable; Roman has no zero).

Hachukma Digits (0–9)

The Hachukma script has its own set of numeral symbols. The table below lists the digits (the individual symbols) 0 through 9, showing each Hachukma glyph alongside its corresponding Arabic digit and Roman numeral equivalent. Since the Roman numeral system has no symbol for zero, that entry is left blank.

Large Kokborok Numerals (10,000 – 10²⁰)

Kokborok has specific names for larger units. These are used in formal counting and traditional contexts. Roman numerals are provided using the vinculum (overline) system. This is a widely used reputational extension, as the official classical system only goes up to 3,999, but the vinculum is commonly accepted for representing larger numbers. The Hachukma digit column shows the number written with Hachukma glyphs.

Unicode PUA Reference

The Hachukma numeral glyphs used in the tables above are mapped to the following Private Use Area (PUA) codepoints. This reference covers the numerals 0 through 9 only.

 U+E000
 U+E001
 U+E002
 U+E003
 U+E004
 U+E005
 U+E006
 U+E007
 U+E008
 U+E009

These codepoints are used with the Hachukma font to render the numeric glyphs. The font maps these private-use values to the appropriate Lekhamung script numerals.
